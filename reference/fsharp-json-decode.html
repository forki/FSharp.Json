FSharp.Json

Decode

Namespace: FSharp.Json

A way to turn Json values into F# values. A Decoder<'a> represents a decoding operation that will either produce a value of type 'a, or fail.

Nested types and modules

TypeDescription
Decoder<'a>

Represents a way of decoding JSON values. If you have a (Decoder>) it will attempt to take some JSON value and turn it into a list of strings. These decoders are easy to put together so you can create more and more complex decoders.
